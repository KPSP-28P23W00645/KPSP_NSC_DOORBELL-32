# KPSP_NSC_DOORBELL (Smart Doorbell-32)

**ออดประตูอัจฉริยะ-32 (Smart Doorbell-32)** คือนวัตกรรมระบบรักษาความปลอดภัยและการควบคุมการเข้า-ออกราคาประหยัดสำหรับครัวเรือนและสำนักงานขนาดเล็ก พัฒนาขึ้นภายใต้โครงการการแข่งขันพัฒนาโปรแกรมคอมพิวเตอร์แห่งประเทศไทย ครั้งที่ 28 (NSC 2026) 

*   **รหัสโครงการ:** 28P23W00645
*   **สถาบันการศึกษา:** โรงเรียนกาญจนาภิเษกวิทยาลัย สุพรรณบุรี
*   **อาจารย์ที่ปรึกษา:** นาย บุญชู กระฉอดนอก

---

## โครงสร้างโฟลเดอร์และรายละเอียดไฟล์ภายในโครงการ

*   `firmware/` : ซอร์สโค้ดภาษา C/C++ สำหรับอัปโหลดลงไมโครคอนโทรลเลอร์ ESP32-CAM (ทำงานเป็น Web Server คอยควบคุม Relay และสตรีมภาพวิดีโอ)
*   `app/` : ซอร์สโค้ดสำหรับพัฒนาต่อยอดแอปพลิเคชันระบบ Android บน Kodular (ประกอบด้วยไฟล์โปรเจกต์ `.aia` และไฟล์พร้อมติดตั้ง `.apk`)
*   `hardware/` : ไฟล์วงจรและการออกแบบแผ่นวงจรพิมพ์ต้นแบบรุ่น **KPSP_NSC_SMARTDOORBELL REV_V1.2** ที่ใช้ซอฟต์แวร์ EasyEDA ในการออกแบบ (ประกอบด้วยแผนผังวงจร, ลายเส้นวงจรพิมพ์, และไฟล์ Gerber สำหรับสั่งผลิตบอร์ดจริง)
*   `cira_ai/` : ไฟล์สคริปต์วงจรทำงาน .npj ของแพลตฟอร์ม CiRA Core สำหรับประมวลผลระบบสแกนและตรวจจับใบหน้าผ่านกล้องวิดีโอสด
*   `docs/` : คู่มือประกอบการใช้งานและการติดตั้งระบบ (ทั้งส่วนเชื่อมต่อสายสัญญาณฮาร์ดแวร์และแอปพลิเคชัน) ในรูปแบบไฟล์ PDF

---

## ฮาร์ดแวร์หลักที่ใช้ในโครงการ
1. บอร์ดควบคุมหลัก ESP32-CAM
2. เซ็นเซอร์ตรวจจับความเคลื่อนไหว PIR (PIR Motion Sensor)
3. ชุดล็อกไฟฟ้าโซลินอยด์ 12V (Solenoid Door Lock)
4. โมดูลแปลงแรงดันไฟ XL4015 (Voltage Buck Converter)
5. ระบบสำรองไฟแบตเตอรี่ 18650 UPS Module
6. กล่องกันน้ำพลาสติกฝาใสสำเร็จรูปสำหรับงานไฟฟ้าภายนอก (Weatherproof Plastic Box)
